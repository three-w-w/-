# 梦境
：我做梦被一个神秘组织抓走闯关，组织里有豪华大厅，所有被挑中的陌生人都在期待奖励，可奖励只有成功过关才能获得自由，没能通关就永远出不去。 我们没法提前知道关卡内容，主持人不肯透露详情，只让所有人随机抽签，每个关卡的鬼怪都是这个组织搜罗来的，像贞子、历史上臭名昭著的杀人狂都会出现在关卡里。我上次闯关遇到一个可怜女鬼，不算吓人，侥幸逃出来了，昨晚这个组织又把我抓回去闯关，这次的关卡恐怖又催泪。

这个关卡的故事主角是偏远乡村里一个极有才华、诗作动人的少女。她从小被困在落后村子，长久被贫穷、疾病、旁人偏见压迫，就连本该疼爱她的奶奶，也是压迫她的人之一，她这辈子最大的心愿就是逃离这里。梦里故事前半段温和好笑，后半段剧情太过痛苦惊悚，我直接在梦里快进看完结局：少女和她的弟弟妹妹全都惨死，下场十分凄惨。

村子通往外界只有一条路，中间隔着一条宽阔大河，村民凿出厚重粗糙的大石块，打磨成宽大石板铺成过河通道。我走到河边，一开始小心翼翼踩石板，担心石板松动，走了一段发现安稳无事，便快步往前走，猛然发现整条石板路底下，全是少女的尸体。河面有多宽多长，她的尸身就有多宽多长，厚重粗糙的石板全部重重压在她身上。

河道侧边倒放着少女的头颅，摆在一旁，摆在我面前的是两道选择：要么踩着她的头颅过河，要么踩着姐弟三人残缺的身体往前走。我特别害怕人头，也恐惧姐弟亡魂会伤害我，咬牙选了踩姐弟三人的尸身渡河。可刚落脚，惊悚的一幕出现了：少女的鬼魂缓缓抬起手，她弟弟妹妹也一左一右同时举起手，我当场吓得以为自己要丧命，会变成第四具尸体躺在河边。

可转瞬我才看懂他们的用意，他们抬手是为了托住我，方便我顺利过河。这几只鬼明明凄惨又恐怖，却格外催泪，他们早已死在这片土地，再也走不出去，是想托举我离开，盼着我能活着出去，好好生活。

等我踩着尸身渡过整条河，对岸是一面极高的堤坝，必须走台阶才能爬上去，我瞬间绝望，以为自己会和少女一样，永远困在这里逃不出去。堤坝边躺着姐弟三人残缺的上半身尸体，肢体不全，缺了胳膊。 我过河全程都要踩在少女的尸身之上，心里格外难受，后来才明白她的尸身为何绵延整条河道——那长长的躯体，承载的是她一辈子无尽的苦难，和她毕生渴求自由的心愿，整场梦境看完，我又恐惧又心酸。

# 第一章：又来了（自由的旅人）

旧书店在一条窄巷子里。门面小，招牌上的字褪得只剩轮廓。推门进去——头顶的铃响了一声。没人。书  架从地到顶，过道只够一个人侧着身。

手指从书脊上划过去，旧书脊上的烫金磨得只剩凹痕。抽出一本，封皮起了毛。翻开，手指按在书页上——按了个空。

一扇金色的门，兀的出现在眼睛。店里的光还在，打在门框上，金色上面浮着一层灰白。

指头上还有灰。手心开始湿了。和上次一样——上次在曼谷，考山路，手里还捏着一张明信片。我把手上的汗在裤子上蹭了一下。门缝底下漏出光来——黄的，带一点暖，但暖得不舒服。里面有人，脚步隔着门听起来很远，像在另一个房间，又像在很深的地下。

没急着推门。上次推太快了，进去之后才后悔——“门外面多站的那几秒，是最后一个能自己选节奏的时刻”。

门自己开了。

大厅比我记忆中大。地砖是米白色的，接缝看不太清，像一整块石头磨出来的。看不见灰，也不见人留下的痕迹。顶上吊着灯，很多盏，光很匀——我环顾都不见影子在哪一侧。空气里有花香，闻不出是什么花。这个味道上次也有。

人不多。

七八个。也许是九个。我没有数。

有人站在角落里，背靠着墙，两只手插在口袋里——他一直在换脚。每换一次鞋底在地砖上磨一下，声音很轻。大厅太安静了，那个声音一直在。有人盯着天花板，一动不动。有人反复摸口袋，摸完左边摸右边，摸完了又摸。没有人在接耳，没有人认识彼此。上次也是。

上次我在大厅里想过找人说话——找个人一起，至少交换名字。后来那个人没出来。

主持人进来了。也可能是本来就在。

声音先到。没有特征。每个字都像被熨过一遍。我找他的脸，找到了。看完了就忘了，存不住。这里面的记忆一直是留不住的像做梦，唯有关卡能让你......

「欢迎回来。」

他对着所有人说。眼睛——如果他有眼睛——没有落在任何一个人身上。

「规则不变。抽签决定关卡。过关，自由。没过——」他没说完。不需要说完。上次活下来就知道。没过的意思就是变成关卡的一部分。和那些鬼一样。吊死的，溺死的，含冤而死的人。哪天你的名字也会出现在签单上，被别人抽到。

他轻轻侧了一下,签筒出现在房间中央。在胸口的高度，有个暗红色的木头，签条从里面探出白边。

“从左边开始。”

我的位置在中间偏右。前面的人一个接一个走上去。我看着他们的后脑勺——有的发际线在往下滴汗。有人抽完往回走，步子比去时小，眼睛不看任何人。有人在伸手的那一下肩膀往上耸——上去，没下来。

前面还剩两个人。一个人的后背在出汗。衬衫贴在肩胛骨上。

我手心已经湿了。上次抽签也是。签筒里的签你没办法挑——你只能抽最靠外那一支，但最靠外那一支每次都不一样。

轮到我了。

签筒比看着要沉。手伸过去才感觉到——它在微微往下坠，像里面有东西在呼吸。我告诉自己别想，别想，抽最靠外那支。

纸很薄。带一点潮。

翻过来。

**阿苇。**

名字下面一行字，手写的，每个笔画都很慢，很重：

**从她的心愿里走过去。**

我把纸条攥在手里。纸片边缘割了一下手心——不疼，但很清晰。

门出现了。

不对，是大厅的一面墙消失了。刚才还在的米白色墙面，上面的光还在，现在什么都没有了。外面不是走廊——是一条土路。

土路两侧是灰墙。墙根长着青苔，青苔颜色很深，像湿了很久。更远处是山。四面都是山。天空被切成一块不规则的形状。

其他人在往自己的方向走。每个人的门开在不同方向。没有人互相看。你抽的签就是你一个人走。

我站在门口。

脚下还是地砖——米白色，光滑，硬。但土路就在一跨之外。能看到路面上的小石子，能看到黄土干裂开的纹路。两种地面并在一起，中间没有门槛，没有过渡，什么都没有。

风吹过来。不是大厅里的风。有泥的味道。有烟的味道。

站得有点久，好似有股力量，我迈出了第一步。腿没抖，手还在抖。脚底下——硬的光的变成了松的粗的。

身后的门消失了，就身下我自己

“静。

太静了。”

没有鸟叫，没有虫鸣，没有风穿过任何东西的声音。只有鞋底擦过土面——我走一步，它响一步。

巷子拐了个弯。

土屋。屋檐下挂着干辣椒，已经褪成了枯色，但还在。屋里有火，烟从瓦缝往上冒。

我停了一下。

烟在冒，没有味道。

村子不是废弃的。

是刚被丢下的。

# 第二章:不会飞的鹤

> 蒲苇韧如丝,磐石无转移。


巷子走到头,村子呈现出来了。

一片土屋挤在山脚下,朝向同一条河。河在村子另一头,青灰色,宽,水面不响,那种静是凝固的，等人来打破。河上铺着石板,一直通到对岸。

屋檐下挂着去年的燕子窝,空的。地上有扫帚扫过的痕迹,扫帚还在墙边靠着。有一家的灶台还在冒烟，细的、直的,往上走到一半就散了。

我往烟的方向走过去。

门开着。门框上贴着一张褪色的红纸,上面的字认不出了。也许是福字。已经和水渍一个颜色。

进门之后闻到的是灰,不是灰尘--是烧过柴以后那种冷掉的灰。灶台里的火还在,很小,红得只剩一层。灶台旁边一张床,床板上的被褥叠得很整齐。桌子,桌上有些东西。

墙上贴着一张奖状。

奖状的纸已经泛黄了,边角卷起来,但没有撕。上面印的字还很清晰--全县小学生诗歌比赛,一等奖。下面盖着章,红印泥,褪色褪到变异成粉色。章上面写着一个名字:

**"苇。"**

这个名字是我在纸条上看到的那个。字的笔画都是硬的:"苇。"

奖状下面有一行铅笔字,写在墙上。小学生写字,一笔一划,压得很深:

**"山是牢,河是锁,我是一条不会飞的鹤。"**

鹤字写得最大。比前面所有的字都大。好像写到这里铅笔顿了一下,然后用力。

桌上摊开着一本作业本。封面已经磨得看不太清原来的颜色。旁边一支铅笔,笔头秃了,没有被削过。还有一个碗,扣着放的。

我把手伸过去,上次关卡里,不该碰的东西碰了以后出了事。但这个村子不一样,不像是要害人的,或许只是想死的我忍不住想看一眼。说不清,反正管它的呢,我总是我有种莫名其妙的豁达。东西全摆在那里,没人收拾的静。

翻开了。

第一页,第一首。

**《鹤》  
天上一只鹤  
叫了一声  
我抬头看  
飞过去了  
没有落下来**

字迹和墙上的一样,看二页。

**《弟弟》  
弟弟偷了锅里的红薯  
烫得左手换右手  
边跑边往嘴里塞  
我假装生气  
其实是想  
剩下那个红薯给我**

这一页的铅笔轻了一些。最后一行--"剩下那个红薯给我"--旁边有个用橡皮擦过的痕迹。她改过。

再翻。

**《月光》  
半夜醒来  
弟弟的脚抵着我的背  
妹妹在梦里笑了  
窗外的月亮  
照在碗里  
碗是空的**

到这里铅笔又重了,和前面第一首的时候一样。

再翻一页,还有。

**《问》  
奶奶说山那边还是山  
我不信  
我想去看一眼**

这一页后面,还翻了一页,是空白的,再翻--还是空白的。作业本只用了四页,第一首到第四首,然后没有了。

手指摸到封皮夹层里有一张叠过的小纸片，我赶紧抽出来。

“明天我们走。”

铅笔，不是阿苇的字。更小，更歪。翻过来——背面是药方。大人的笔迹，几味药，后面跟着钱数。

我把纸条折回去，按原来的印子，放回夹层。

合上作业本,手放在封面上,封面粗糙,磨得起了毛。

屋角有声音,从外面来的。很近,鞋底擦过土面,一下,停了,又一下。

我走出去。

对门是一间更小的土屋。门开着,里面很暗,门口坐着一个人。

老妇人,背弓着,头发全白了--不是很长的白发,是贴着头的,稀的。她在擦一双布鞋。鞋很小,小到她的手掌刚好能托住。鞋面已经磨得发毛,手上不停,擦完左边擦右边,擦完右边又回到左边。她的眼睛不看鞋，也不看我。

"请问--"

她没有抬头。

"这里有个叫阿苇的女孩--"

她的手停了。鞋还托在手掌上。

"在河里!!!"

声音不像老人的声音,不哑,不颤--是从很深的地方提上来的。

"河里"

"你到河边就知道了。"

她低下头。继续擦鞋。手上的动作和刚才一模一样,好像中间没有停过。

我还想再问,但她的身体已经缩回门框的阴影里了,像一扇门关了一下。不是真的关了--只是她不在了。

我退了一步,脚后跟碰到硬土。空气里还是灰的味道。

远处河水的颜色更重了,从青灰变成铅灰。石板路通到河边,一直铺到水面以下--不对,不是铺到水面以下,是水面漫过了石板。石板一直在底下,水和石板之间没有缝隙。

巷子那头,老人却还在门口。隔了一条巷子,看不清她的手,只看身体在动,很慢。还在擦那双鞋。

烟还在冒。还是细的,直的,往上走到一半就散了。

没有味道。

# 第三章：沉下去

从老人门口走回来以后，我在巷子里站了一会儿。

河水还在远处。颜色已经从铅灰沉到了更深的一种颜色，像是有东西从河底泛上来了，又像是天色压在河面上，把河水压暗了一截。对岸的堤坝已经不太看得清了，只剩下一条淡淡的轮廓，比天稍微深一点，快要化进夜色里去。

烟还在冒。我辨认了一会儿，才确定是哪间屋子。走回去。其实腿有点不想动，但站也没别的地方可站。这村子就这么大，除了那间屋子和那条河，没有别的什么了。

奖状还在墙上。作业本还在桌上。但这一次进门，我自己都没意识到——我在低头看。目光落在桌下的位置，而不是平视的地方。大概是我心里已经知道，桌上和墙上的东西我已经翻完了，剩下的东西都在更低的地方。

灶台下面堆着几个空药罐。我数了数——三只，也可能是四只，最里面那只压在阴影里，不太好确定。罐口一律朝下扣着，最底下一只裂了一道口子，从罐口一直裂到底。药渣早就干了，结成一圈黑褐色的硬壳，贴在罐子内壁上，看起来放了好久了。我蹲下来，凑近了看。其中一罐的内壁上有一圈白线——水位线。药水烧干以后留下的痕迹，像年轮一样。有人守着这罐药守了很久，一直守到水烧干。

灶台旁边放着一口木箱。盖子没合严，露着一条缝。打开看——衣服。三件小孩子的衣服，叠得整整齐齐。最上面一件的领口脱了线，脱线的针脚还在，有人补过，后来又脱了。下面一件的袖子上打着补丁，原来的布是蓝的，补上去那块是灰的，针脚很粗——不是裁缝缝的，是家里人自己缝的，缝的时候大概光线不好，针脚走得不太稳。最底下一件，背后有一块洗不掉的黄渍，颜色已经渗进布纹里了，边缘是模糊的，不知道是药还是别的什么。时间太久了，分不清了。

我把衣服叠回去，手碰到箱底——凉的。那种铁皮的凉，像很久没有人碰过的东西。

站起来的时候，眼睛扫到墙上。奖状旁边还有一张。刚才只注意到那张一等奖，没看到旁边还有一张。也是奖状，被撕过——从左上角往下扯，扯到一半不知道为什么停了手。撕开的那一截耷拉下来，像一小片枯叶挂在墙上。我伸手翻过来。全县小学生诗歌比赛，三等奖。同一年的。一等奖和三等奖是同一年。她把最好的和最差的贴在一起，中间隔了一道她自己撕开的裂口。为什么要撕？撕到一半为什么又停了？说不清楚。大概有些问题是没有答案的。

奖状旁边的墙上，有铅笔印子。横的，竖的，没有规律。像是有人闭着眼睛在上面发泄什么，画了一通，然后停了。

光又薄了一些。外面的天在往下沉。

床，枕头下面露出一个角。

抽出来。又一本作业本。比桌上那本更旧，封面卷得更厉害，边角有一排牙印——不是咬破的，是牙压过以后留下的印记。有人趴在床上，一边写本子，一边把本子的角咬在嘴里。我能想象那个画面——一个小女孩趴在枕头边，铅笔在本子上沙沙地走，嘴巴咬着纸角，不让自己出声。

翻开。字和桌上那本不一样了。不是一笔一划写的。铅笔斜着下去，写完一个字拖出一道尾巴。用力不均匀，有的字重得几乎戳穿了纸，有的字轻得只剩铅笔尖刮出来的印子，要凑得很近才能辨认。

《鹤·二》
那只鹤再也没有飞回来
也许它知道
落下来就飞不走了

翻过去。

**《病了》
妈妈把最后的药给了弟弟
弟弟好了
我还在咳
妈妈说你大了 能扛**

铅笔在这里轻得几乎看不见。「能扛」这两个字——没有被用力写。像是写到最后一个字的时候，已经没有什么力气了。铅笔尖在纸上拖了一下，留下一条淡淡的划痕，就停了。像是写到这里就不想再写了。也可能是已经没有力气再写下去了。

翻过去。

**《奶奶说》
奶奶说等春天就好了
春天来了
又说等秋天**

翻过去。最后一页。

铅笔用力压在纸上，但字没有写完。

**“河的那边
我想去
我”**

「我」字后面什么都没有。铅笔停在那里——不是停下来想一想再接着写，是把笔放下来以后再也没有拿起来的那种停。

再翻，空白的。再翻——还是空白的。和第一本一样，写了四首，停在一个没有写完字的句子里。

但翻到最后一页的时候，我手指的感觉不太对——指尖滑过的地方，有一小片忽然没有纸的阻力了。我拿起来看，缺了一个角。从右上角往下撕了一小块，留下一条毛边。被撕掉的那一小片不知道去了哪里。

缺角旁边的纸面上，有铅笔擦过的痕迹。先用力写了，然后用橡皮反复擦过，纸面的纤维都擦毛了，铅笔粉被擦走以后，留下了一层灰蒙蒙的印记，像褪了色的记忆。我把本子侧过来，让门口最后一点光斜着打在上面。

隐隐约约能认出几个字。

**我等——**

底下还有一个字。被擦得更彻底，纸面被橡皮磨得都发亮了。我眯着眼睛看了很久。像一个「走」字。我不太确定。也可能是我想多了。就这些。写上去，又擦掉了。只留下「我等——」和底下那个被擦得几乎看不出来、但好像又确实存在过的字。我不知道她为什么要擦掉。也许她自己也不确定了。

我合上本子。封面上的牙印对着我的手指。

药罐里的水位线。衣服上的补丁。被撕了一半的奖状。缺角纸上擦掉的「我等——」。我想起桌上那本作业本。走过去，翻开封皮，手指伸进夹层里——那张叠过的纸条还在。抽出来，展平。

**明天我们走。**

五个字。铅笔写的，比阿苇的字小得多，也歪得多。字与字之间不在一条直线上，上下左右地歪着——像是趴在炕沿上写的，一边写一边回头，怕被人发现。不是她的字。是弟弟的。翻过来，背面是药方。大人的笔迹，潦草，几味药的名字后面跟着钱数：三钱、两钱、各若干文。拿这张纸去镇上药铺，称几服药，然后回来。这就是「明天我们走」的全部意思——不是翻过山的那一边。是去镇上买药。她弟弟能想到的最远的逃亡，到这里就到头了。

我把纸条叠回去，按原来的印子，放回夹层。

药罐里的水位线。衣服上的补丁。被撕了一半的奖状。缺角纸上擦掉的「我等——」。第三本作业本上，最后一个字是「我」——她写下自己的名字，没有继续。

她们没有逃走，她们没有走成。

灶台和床之间的地上放着一只碗。就是桌上那只扣着放的碗。我弯腰把它翻过来——翻过来之前我就大概知道里面是什么了。粥，干成了一层浅黄色的硬壳，沿着碗壁裂开，裂纹从碗沿一直延伸到碗底，像旱地里干裂的泥土。筷子斜插在里面，筷子头埋在粥壳里，筷子尾翘着。不是吃完了放下的。是吃到一半放下的。有人放下这碗粥以后再也没有回来。

桌子底下有一个布团。我弯腰捡起来——是一个布娃娃。用一块破布扎的身体，没有手没有脚，只有一团身体和一颗圆形的头。眼睛是线绕出来的，绕了三圈，左边的比右边大。不是故意的，这我看得出来。绕到左边的时候手停了，想起了什么事，结果多绕了一圈，回神过来也懒得拆了。嘴巴用蓝色的笔画了一个弧线，有点歪，笔迹歪到一边脸颊上去了。有人给它画了一张脸。让它看起来像在笑。

布娃娃旁边躺着一个弹弓，架子是树枝削的，削得不平——不是一刀一刀慢慢修圆的。一刀砍下去，停了，再砍一刀。小孩的手劲，没什么耐心。橡皮筋从中间断了，断口是毛的——不是拉断的。橡胶老化了，自己裂开的，时间把它磨断了，不是谁的力气。

桌子腿靠墙的地方垫着一张纸，作业本的封皮。我抽出来，翻到背面——画着三个火柴人，手拉着手。最高的那个旁边写了两个字：姐姐。铅笔压得很重，写完以后又描了一遍，怕那两个字会消失似的。和墙上那个鹤字的力道一模一样。

我把画翻过来。正面是封皮——全县小学生诗歌比赛，一等奖。原来这张封皮就是从墙上的奖状旁边撕下来的。撕下来以后，被她拿来给弟弟妹妹画了一幅画。三个手拉着手的火柴人。中间那个画了好几遍。

我准备把画放回去。蹲下去的时候，余光扫到了墙根。

那行铅笔字还在墙上。

**山是牢，河是锁，我是一条不会飞的鹤。**

字的高度齐腰。我蹲下来以后，视线比那行字还要低。我本来打算站起来，但不知道为什么，眼睛往下面多看了一眼。

字下面的墙缝里——灰墙和地面交接的那道夹缝——有几道白色的划痕。不是铅笔。是硬物在墙面上划过去留下的印记。我定睛看了两眼。不止一道。我侧下身子，把脸凑近，几乎贴到地面上去了。

不是随便划的。是指甲划出来的。有人蹲在这里，用指甲在墙缝里用力划的。不是一口气划下来的。划一道，停一下，想一想，再划一道。我借着最后那一点光仔细辨认。

三道白痕。

**——不走了。等——**

「等」字的最后一划拖得很长。像是写到这里停住了。手没有抬起来，指甲搁在墙上，搁了很久。为什么是不走了？是不想走了，还是走不了了。没有答案。

我蹲在那里看了很久。站起来的时候膝盖酸了，腿也麻了。

天已经黑了。屋里什么也看不清楚了。灶台里的火不知道什么时候灭的，连余烬的那一点红光都没了。烟也停了。

我从屋里走出来。

外面比屋里只亮一点点——那种天刚黑透的时候，天还没有完全黑死，还有一丝暗蓝色的光。河水的颜色已经完全化进夜色里了。堤坝的轮廓还在——比天略微黑一点的一条线，横搁在远处。石板路从村口伸出去，一直铺到看不见的地方。

明天要走那条路。

明天要过那条河。

今晚就在这间屋子里等天亮吧。

# 第四章：过河

天亮了以后我才发现自己没有睡着过。

眼睛闭了一夜，但耳朵一直是醒的。一整夜——没有风声，没有虫鸣，没有木头在夜里收缩的响动。只有我自己的呼吸，还有心跳。后来我憋着气听——什么都没憋出来。这村子死得像一个密封的盒子。

最后我睁开眼的时候，天已经亮了——不是一下子亮的，是一点一点把黑暗稀释掉的。窗子外的颜色从黑变灰，从灰变白。

我在床边坐了很久。

灶台里的火已经冷透了。灰是白的。我摸了一下——凉的，摸不出任何温度。

桌上那本作业本还在。还在翻到最后一页的位置——铅笔停在那里，那个没有写完的「我」字。碗还在原地扣着。什么都没有变。我手伸出去，想把那页翻过去，手指碰到纸的瞬间又缩回来了。

没碰。

把地上的画捡起来——三个手拉手的火柴人，姐姐那两个字描了两遍。我叠好，压在枕头下面。不知道为什么。可能是不想让它在地上。

纸条还在口袋里。那条纸边割过的地方还刺着。

**从她的心愿里走过去。**

我把它展开，念了一遍，折回去，放回口袋。

手心在裤子上蹭了一下。干的。

走出来。

天是灰白的。和昨天一样，和刚进村的时候一样。但河水不一样了——颜色比昨天更淡，说不上来是为什么。好像一夜之间水里多了什么东西，稀释了原来的深色。又好像河底有什么东西在往上升，把水的颜色顶淡了。我站在那里看了一会儿，河面很平，平得不正常。连风都没有给它一丝皱纹。

石板路从村口伸出去，一直铺到对岸。每一块都粗凿的，边缘不齐，带着石头刚被凿开时的棱角。这条路上的石头没有人走过，没有被鞋底磨过。凿痕还在。我记得第二块石板上有一道斜的凿纹，还留着。

走向河边。

前几步踩在土上，土是松的，鞋底陷下去一点。然后是第一块石板。

我踩上去的时候，脚下的触感变了。从松到硬，从暖到凉。石头吸了一夜的凉气，隔着鞋底传上来，凉的。

往前。

每块石板之间隔了大概半步。一开始很正常——脚踩下去，石头不动，鞋底在粗糙的表面上轻轻擦过。第二块。第三块。第四块。但走到第五块的时候，有什么东西开始变了。

石板之间露出底下的东西。

不是水。是颜色不对。水的颜色应该是深的——铅灰、墨绿、或者被天空的颜色染透。但石板和石板之间的缝隙里露出来的，不是水。看起来比水实，比水浓，有厚度。

我停下来。

石板开始稀了。不是一块接一块——是隔一段才有一块。走到第七块的时候，只有一半的脚踏在石板上，另一半悬空了。我低头看。

石板和石板之间露出的，是人肉的颜色。

不是红的。是泡了很久的水以后，皮肉变成的那种颜色。灰白中泛着淡青，边缘被水泡得起了一层薄薄的皱皮。

前面没有石板了。

不是没有路了。是石板不再一块一块铺在路面上——石板变成了嵌在路里面的东西，零散地嵌在人的身体上。前面那段——石板嵌在什么圆的东西上，像是肩膀。再往前一块——嵌在腰上。再往前——嵌在大腿上，石板的边缘把肉压出一个凹陷，骨头撑在石板底下，角是凸出来的。

血肉，清晰可见。

不是一小段。不是几具。是整条河——从头到尾，从脚下到对岸，一节一节连着。人的身体铺满了整条河道。路面就是肉。石板嵌在肉里——腰上嵌着一块，大腿上嵌着一块，肩膀那块把骨头顶得变了形，露出来的骨茬是白的。水从身体和石板之间的缝隙流过，碰到石板的时候起一圈很小很小的水纹，然后继续走。

两侧，是滴着墨色液体的头颅。

沿着河道排过去的。不是漂浮在水面上的——是侧放着的。每颗头都侧过来，面朝对岸。有的眼眶是空的，空的眼眶里什么都没有，干干净净。有的眼睛还在——睁着的，眼睛上蒙着一层灰白的东西，像在水里泡久了的鱼眼。眼珠没有在看什么。它们只想朝着对岸。

三颗。一大两小。

最小的那颗头发还在——贴在额头上，湿的。头发不长，细软，打成了几绺贴在面颊上。面朝对岸，额头上一缕碎发没有贴住，在水面上微微地漂。像生前那样。像她趴在桌上写诗时一样。

我站在最后一块完整的石板上。

前面就是她的腰。那块石板嵌在她的腰上——石板和她的腰之间没有缝隙。腰侧面的皮肤被石板的边缘压得微微凹进去，翻出一层灰白色的皮肉，像肉被石头压扁了以后慢慢地长回去了。石板旁边，没有石板的地方，露着她的身体——灰白的，被水泡透了的，但轮廓还在。腰的曲线还能认出来。再往前，大腿。她生前大概是瘦的——大腿的骨头支出来的形状很清晰，膝盖骨圆而小。

要过去。要么踩头颅，要么踩尸身。

她的头在左侧。弟弟妹妹的头在右侧。三颗头，紧贴着石板的边缘，面朝对岸。她额头上那缕碎发还在水里漂着。

我低下头看自己的脚。

鞋底还有昨天沾的土。土干了，裂成细纹。

要过去。

我怕和人对视。看眼睛会让我想起她们活着的样子。身体只是一段一段的肉——它们不会看着我。踩了身体。

第一脚踩下去的时候我没有闭上眼睛。不是不想闭。是闭不上。

脚落在阿苇的腰上——那块石板旁边，没有石板的地方。脚底直接碰到她的身体。

软的。

底下有骨头。

软的。泡了很久的水，皮肉已经失去了弹性，踩上去的时候脚底微微往下陷，陷到一个深度就停了——被骨头挡住了。她腰侧的皮肤贴着我的鞋底。我不知道那是什么触感。冷的。软的。沉的。

脚没有抬起来。

脚下的身体动了。

不是身体。

是手。

从水面下。一只。从河面下抬起来。先是手腕，然后是整只手。

水从她的手指缝往下淌。一滴一滴。不是涌出来的——是顺着手指流下去的。手抬起来的时候水面分开，水从指间滑落，在阳光下亮了一下，然后滴回河面，发出一声很细很小的响。

不仅一只手。两只。三只。

阿苇的右手从腰侧伸出来。弟弟的左手从旁边伸出来——手很小，骨节还是小孩的手。妹妹的右手从她的另一侧伸出来——最小的一只，手指虚握着，像抓着什么看不见的东西。三只手。同一个方向。朝上。朝我。

水面还在往下淌。从她们的指缝。从她们的手腕。

滴回河里。

我动不了。

腿锁住了。膝盖不会弯了。脚还被钉在那块身体上。手也不会动了。呼吸停了。肺里有一口气——吸进去了，出不来。那口气堵在胸口，压着。

三只鬼手从河面下抬起来。

水还在从她们指缝里往下淌。

没有停。

没有加快。

就是那样。

等着。
# 第五章：你出得去吗

三只手在面前。

从水面下伸出来的。阿苇的右手。弟弟的左手——很小的手，指节还是小孩的样子。妹妹的右手——最小的那只，虚握着。三只手。在我面前。半空中。

水从指缝往下淌。一滴一滴。滴回河里。水面被砸出一个一个很小的圆，然后平了。没有停。没有加快。

我动不了。

脚还踩在她身上。软的。底下有骨头。手在面前——阿苇的右手离我最近，指尖几乎碰到我的膝盖。

手没有动。

就停在半空。三只。没有抓。没有往下拖。没有把我往水里拽。水从指缝往下淌。手停在半空。

等着。

我不知道自己站了多久。可能是十秒。可能是两分钟。时间在那段路上是乱的。我只知道自己站在那里，脚底下踩着她的身体，面前悬着三只淌水的手，我和她们之间隔着一段空气，这段空气像被什么撑住了——软的，沉的，有温度的。

后来是腿先动的。不是我想动，是站不住了。腿开始发抖，抖到膝盖撑不住了，脚提起来往前落。

她胸口。那最后一块石板嵌在她的左胸侧面，石板的边缘压住了一侧肋骨——她的胸骨从皮肤下面支出来，又小又薄。我的脚落下去的时候，踩在她锁骨下面那块凹陷的地方。

手跟过来了。

阿苇的右手。从腰的位置——不知道什么时候——移到胸口。在我脚底下。我的脚落下去，手就在那里。不是托——是垫。手背贴着河面，手掌朝上，手心对着我的脚底。

我停了一下。

那瞬间我脑子里什么都没有。没有恐惧。没有思考。只是一种感觉——脚底下有东西。不是石头，不是土。是手。软的。活的。在下面。

我又走了一步。踩在她左胸肋骨上——一根一根，隔着一层皮肉，像踩在一架琴键上。弟弟的手从旁边跟过来。妹妹的手也从另一侧跟过来。三只手，我走一步，她们移一步。每一步脚落下去，手就在下面。

不抓。不拖。垫着。

水面在脚下。有时踩下去水会没过鞋面——凉的，那种河水浸透鞋面的凉——但脚底下总有手。手从水底下顶起来，把我往上撑。不是用力。只是在那里。手心弓着，贴着鞋底，像一只碗。

我走得很慢。不是不想走快。是每一步都需要时间。踩下去。等手跟过来。确认脚底下是实的。再走下一步。

走到河中间的时候腿停了。

不是走不动了。是从踩上第一块石板开始，这只手就在底下。从跨过那扇门开始——从抽到那张纸条开始——就在底下。那个念头不是突然涌上来的。它一直在，只是现在才浮到水面上。

我低头看。水是透明的。不是清——是透明到能看见底下的一切。她的脸在水底下。头发散开了，在水里慢慢地漂，一绺一绺的黑发，像在水里画着什么，画完又散掉。她的眼睛是闭着的。不是死了以后那种闭——是睡着了那种闭。嘴唇微微张开，像还有什么话没有说完。

面朝对岸。

从看见她的脸那一刻起——我知道了。她的头也在朝着对岸。和她的诗一样。和那只鹤一样。

我的眼泪掉下来了。自己都不知道什么时候开始的。不是哭。是一种东西从身体里往外涌，没有任何声音。眼泪掉在水面上，和那些从她指缝里往下淌的水混在一起。我继续走。

腿踩过她的肋骨。踩过石板嵌得最深的那一块——那块嵌在她骨盆上，边缘被压了很久，肉和石头长在了一起，分不清哪里是石头哪里是她。踩过大腿——大腿内侧的皮肉最薄，骨头直接顶在脚底。然后脚下的身体到头了。

前面是淤泥和碎石。对岸。

脚踩在土上的时候腿一软。没有摔倒——手撑在地上。土是冷的。和刚进村的时候一模一样。那种土腥气也是。安静也是。

回头。

那条河。水面下的身体从头到尾铺满了整条河道。石板零散地嵌在她身上，嵌在腰上，嵌在肩上，嵌在腿上。水在流——很轻，很慢。碰到石板边缘的时候发出一声很小的响动——噗，像什么东西软软地擦过去——然后继续走。她的手还在水底下，还在抬着。弟弟和妹妹的手也在抬着。三只手，从我走过的地方一直延伸到河心，还在那里。

那条路我走完了。

她还在里面。

我蹲下来。河面很静。水流得很慢。一绺黑发在水底下慢慢地漂。

我蹲在那里看着水面。看了一会儿。她的头发动了——被水流带着——往对岸的方向。从头到尾，从第一块石板到最后一颗头，所有的人都在看着对岸。她也是。从头到尾都朝着一个方向。

望了一眼堤坝。

从河岸往上，石阶一级一级。不是规则的——每一级的高度不一样，有的高到膝盖，有的只到脚踝。台阶两侧的土被踩实了，踩成了硬硬的褐色。

看不到顶。能看到的只有第一级——还有第一级台阶两侧的什么东西。

三具上半身。

残破的。缺了胳膊。肩膀以下空着。断口的地方不是平的——像被什么钝的东西砸断的，骨头茬子露在外面。手在河里。我知道手在河里——我刚刚从那条路上走过来。那是她的手。

最小的那具。肩膀的宽度不到两只手掌并排。侧躺在台阶旁边。面朝来时的方向——面朝河，面朝姐姐在水底下的手。妹妹。她侧躺着，断掉的肩膀斜对着我。头歪在一边，额头抵在台阶上，像睡着了。

我看了很久。妹妹的头发贴在额头上。干的。不像在水里那样湿着。她的睫毛还在。脸上的皮肤——虽然灰白——但还能看出轮廓。很小小的一张脸。

我弯下腰。伸出手。在自己的手指碰到她之前，停住了。

手缩回来。

往上。

每级台阶都很高。登一级膝盖就沉一分。大腿酸了。后颈在出汗。我没有数级数——头几下还数着，后来就忘了。只有膝盖在响。只有呼吸越来越粗。

爬到顶的时候，视野突然开了。头顶上不再有山围着。

能看见远处的云。云在天上慢慢地走，走得极慢，慢到你不盯着它看，就不知道它在动。风迎面吹过来——不是从山谷里压过来的那种风，是空旷的、平缓的，从很远的地方一路吹来的。风里有一股味道。不是花香。不是灰。不是水。是活的味道。草、泥土、还有被太阳晒过的东西上散发出的那种——暖和的味道。

风一吹，我猛地吸了一口气，吸得很深，好像这辈子第一次吸进去一样。

回头看那条河。

从堤坝顶上往下看。整条石板路横跨水面——一段一段的，中间几块石板嵌在身体上，旁边几块空着，露出灰白的身体。水在板与板的间隙里流。底下绵延的尸身——从头到尾，从这一头到那一头。腰、胸、肩、腿——每一段都在。最小的那颗头还在最前面，额头上的发还在漂。

河有多宽，她就有多长。

风从背后吹过来。我的头发朝前面飘。衣领在翻，贴在脖子上又翻开。

身后有什么东西。不是声音。是一种感觉——身后的空间不一样了。不是墙。不是空地。是很空的东西，空得发亮。

我转过身。

门。

就站在那里。和进来的时候一模一样的金色。一样的高度。门缝里漏出光——白色的，普通的。不是那种会灼伤眼睛的白。是很普通的白色光。日光灯那种白。可以走出去的那种光。

我站在那里看了很久。门把手是金色的，没有灰，没有人碰过的痕迹。

手伸出去。在碰到之前停了。

风还在吹。头发在脸上扫。远处的云还在动。味道还在。

她闻到过这个味道吗。她这辈子闻到过风从这么远的地方吹过来的味道吗。

手握住门把。

金属的凉——不冰，是那种被空气中的温度浸透了的金属，不冷不热。手心贴在上面。

可以出去了。

我记得上次在曼谷。那扇门推开以后我站在外面的街上。考山路。灯还亮着。有人还在卖烤串。明信片摊还开着。我站在那里看着那些光看了很久。走回去买了一瓶水。水是凉的。瓶子外面挂着一层水珠。手指碰到的时候才知道自己活过来了。那瓶水——我喝了三口。然后蹲在路边吐了。吐完以后站起来，把剩下的水喝完了。然后走了几步，又站住了。

我松开手。回头看了一眼那条河。

她还在那里。

我转回来。重新握住门把。

推开了。

白色的光涌过来。没有温度。脚踩上去——不是土路了。不是石板。不是身体。是硬的光滑的平整的地。像大厅的地砖。但不是我进来的那个大厅。单纯只是一片白。

身后门没有关上。

我回头看——那条河，堤坝，村子，还看得见。但颜色在变淡。从灰白变成更淡的灰白。不是消失了。是留在另一个地方了。像隔着很深的水在往上看。

我在那片白里站了一会儿。

没有方向。没有墙。没有声音。只有白。

然后。

不对劲。

门没有在我身后关上——但也没有消失。它就那样开着，像一张嘴，但我走进去以后什么都没有发生。没有走廊。没有出口。没有变化。

白。

「我」

声音。

和第一章一样的。主持人的声音。没有特征，每个字都像被熨过一遍。从前面来的。从四面八方来的。从门里来的。从门的方向。他站在门口。金色的门框。门漏出的光打在他身上——我看见了他。和第一章一样，我看清了，然后记不住。脸是模糊的——不是模糊，是看一眼知道那是张脸，但你无法描述它。眼睛的位置、鼻子的形状——存不住。

「一个简单的问题。你才能真的出去」

他站在那里。没有动。光在他身后。

**“她想要什么？”**

风停了。

远处那条河的颜色不再变淡了。整片白都安静了。

**她想要什么。**

这个问题的答案我见过。她在作业本上写过的。墙上的字。奖状上的名字。那个没有写完的「我」字。缺角纸上被擦掉的「我等——」。三只抬起来的手。到死都在垫着别人过河的手。

**她想要什么。**

我闭上眼睛。

嘴里不停地回忆着：

**天上一只鹤**
**叫了一声**
**我抬头看**
**飞过去了**
**没有落下来**

......


(完结。你能出去吗？)
